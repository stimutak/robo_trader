### How It Works

**Detection Strategy (Priority Order):**

1. **Environment Variable** - `IBKR_PORT` if set
2. **Process Detection** - Check for Gateway/TWS processes
3. **Port Scanning** - Check for listening ports (4002, 4001, 7497, 7496)
